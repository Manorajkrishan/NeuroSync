## Features
1. **Avatar Display**
   - Shows emotions (happy, sad, calm, etc.)
   - Animates when speaking
   - Changes expression based on context

2. **Chat Interface**
   - Messages from AI (left side, with avatar)
   - Messages from user (right side)
   - Scrollable message history
   - Clean, modern design

3. **Compact Controls**
   - Input bar at bottom
   - Quick action buttons (Voice, TTS, Camera)
   - Collapsible sections (Voice Notes, Settings)

4. **Smaller Window**
   - Default size: 600x800px (or responsive)
   - Can be resized
   - Mobile-friendly

## Benefits
- **More personal** - Feels like talking to a friend
- **Less overwhelming** - Cleaner, simpler interface
- **Better UX** - Familiar chat interface
- **More engaging** - Visual avatar adds personality
- **Easier to use** - Everything in one place
